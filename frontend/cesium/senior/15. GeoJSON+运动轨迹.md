

### 概述
前面学习过全栈GIS开发，这里通过QGIS绘制经纬度点，然后通过geoserver提供对外的服务，从而转换成运动轨迹所需的数据集。



```javascript
// 注意添加跨域处理
const response = await fetch('/geoserver/ne/ows?service=WFS&version=1.0.0&request=GetFeature&typeName=ne%3Adsn-point&outputFormat=application%2Fjson&maxFeatures=50')
const data = await response.json()
const drivingData = data.features.map((feature, index) => {
  const [lng, lat] = feature.geometry.coordinates
  const height = feature.properties.height
  return {
    id: index,
    longitude: Number(lng),
    latitude: Number(lat),
    height: 100
  }
})
```
