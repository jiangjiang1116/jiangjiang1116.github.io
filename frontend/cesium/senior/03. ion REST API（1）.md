

### 概述
Cesium ion REST API 是 Cesium 官方提供的 基于 HTTP 的编程接口，用于在程序中管理和访问 Cesium ion 上的资源。通过 API，开发者可以无需手动登录 Web 控制台，就能实现 自动上传、管理、查询和访问资源。



地址：[https://cesium.com/learn/ion/rest-api/](https://cesium.com/learn/ion/rest-api/)



```javascript
// 查询资产
const res = await fetch('https://api.cesium.com/v1/assets/3900873', {
  headers: {
    Authorization: `Bearer ${Cesium.Ion.defaultAccessToken}`
  }
})
const data = await res.json()
console.log(data)
```



```javascript
// 修改资产
const res = await fetch('https://api.cesium.com/v1/assets/3900873', {
  headers: {
    Authorization: `Bearer ${Cesium.Ion.defaultAccessToken}`,
    'Content-Type': 'application/json'
  },
  method: 'PATCH',
  body: JSON.stringify({
    name: 'tentea2'
  }),
})
console.log(res)
```



```javascript
// 删除资产
const res = await fetch('https://api.cesium.com/v1/assets/3900873', {
  headers: {
    Authorization: `Bearer ${Cesium.Ion.defaultAccessToken}`
  },
  method: 'DELETE',
})
console.log(res.ok)
```
