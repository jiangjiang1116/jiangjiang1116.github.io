

### 概述
Cesium 后期处理，是指场景完成正常三维渲染之后，对“整张屏幕结果”再进行一次基于 GPU 的图像处理。



内置后期处理方法：

```javascript
viewer.scene.postProcessStages.bloom.enabled = true  // 开启泛光效果
viewer.scene.postProcessStages.fxaa.enabled = true  // 开启抗锯齿

// 景深效果
const dofStage = Cesium.PostProcessStageLibrary.createDepthOfFieldStage()
dofStage.enabled = true
viewer.scene.postProcessStages.add(dofStage)

// 模糊效果
const blurStage = Cesium.PostProcessStageLibrary.createBlurStage()
blurStage.enabled = true
viewer.scene.postProcessStages.add(blurStage)

// 夜视效果
const nightStage = Cesium.PostProcessStageLibrary.createNightVisionStage()
nightStage.enabled = true
viewer.scene.postProcessStages.add(nightStage)
```
