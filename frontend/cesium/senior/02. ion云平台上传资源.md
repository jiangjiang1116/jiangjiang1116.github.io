

### 概述
Cesium ion 的核心功能之一，就是允许用户将自己的 GIS 数据 / 3D 数据上传到云端，比如：geojson数据或glTF数据，并通过  Cesium.IonResource.fromAssetId() 方法进行调用。



```javascript
// geojson
const resource = await Cesium.IonResource.fromAssetId(3900804)
const dataSource = await Cesium.GeoJsonDataSource.load(resource)
viewer.dataSources.add(dataSource)

// gltf
const resource = await Cesium.IonResource.fromAssetId(3900805)
viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(121.47, 31.23, 1000),
  model: {
    uri: resource,
    scale: 300,
  }
})
```



### 自动转换格式
Cesium支持的数据格式：3DTILES GLTF IMAGERY TERRAIN CZML KML GEOJSON。那么不是这些格式的数据可以通过ion平台进行自动转换，这也是ion云平台的优势，比如：

+ .obj -> .gltf
+ .tif -> .png（IMAGERY）
+ ...
