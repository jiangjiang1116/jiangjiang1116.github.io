

### 概述
Cesium ion 是 Cesium 官方推出的 云端地理空间数据平台（SaaS），主要用于：  

+ 提供 Tokens 的权限管理
+ 托管（Host）地理空间数据资源
+ 在线转换（Convert）各种 GIS / 3D 数据格式
+ 提供可视化创作工具、数据裁剪工具等能力



网址：[https://ion.cesium.com/](https://ion.cesium.com/)



### 内置资源
+ <font style="color:rgb(45, 58, 66);">Bing Maps Aerial（必应地图卫星影像）</font>
+ <font style="color:rgb(45, 58, 66);">Cesium World Terrain（全球地形数据）</font>
+ <font style="color:rgb(45, 58, 66);">Cesium OSM Buildings（OSM 全球三维建筑数据）</font>
+ <font style="color:rgb(45, 58, 66);">Google Photorealistic 3D Tiles（Google写实级三维Tiles）</font>
+ <font style="color:rgb(45, 58, 66);">...</font>



```javascript
const viewer = new Cesium.Viewer('cesiumContainer', { 
  terrain: Cesium.Terrain.fromWorldTerrain({  // 启用全球地形
    requestWaterMask: true,
  }),
})

const osmBuildingsTileset = await Cesium.createOsmBuildingsAsync()
viewer.scene.primitives.add(osmBuildingsTileset)
```

 

```javascript
const viewer = new Cesium.Viewer("cesiumContainer", {
  globe: false,
  geocoder: Cesium.IonGeocodeProviderType.GOOGLE,
})

const googleTileset = await Cesium.createGooglePhotorealistic3DTileset()
viewer.scene.primitives.add(googleTileset)
```
