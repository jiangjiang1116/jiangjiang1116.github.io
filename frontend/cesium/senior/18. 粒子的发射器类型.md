

### 概述
在 Cesium 粒子系统中，发射器（Emitter）负责决定粒子的「出生区域」与「初始发射方向」。不同类型的发射器对粒子的空间分布和效果呈现有根本影响。

Cesium 提供以下发射器类型：

+ CircleEmitter（圆形发射器）
+ BoxEmitter（盒子发射器）
+ ConeEmitter（锥体发射器）
+ SphereEmitter（球体发射器） 



```javascript
emitter: new Cesium.CircleEmitter(10.0),   // 发射半径（米），粒子之间的范围

emitter: new Cesium.BoxEmitter(new Cesium.Cartesian3(5, 5, 5)),  // 可以控制x y z 不同轴的扩散距离

emitter: new Cesium.ConeEmitter(Cesium.Math.toRadians(30.0)),  // 角度
      
emitter: new Cesium.SphereEmitter(5.0),    // 半径
```
