

### 概述
前面学习过 CZML ，它是 Cesium 官方提供的一种  JSON 格式的描述语言，可以用于描述实体集合，当然也可以利用 CZML 实现动画。



```javascript
import point from './assets/czml/point.json'

const dataSource = await Cesium.CzmlDataSource.load(point)
viewer.dataSources.add(dataSource)

dataSource.clock.multiplier = 1
viewer.clock.shouldAnimate = true
```
