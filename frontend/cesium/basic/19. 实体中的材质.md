

### 概述
Cesium 中的材质（Material）用于定义实体（Entity）外观的渲染方式，包括颜色、透明度、纹理、光晕、渐变、动画等视觉效果。

#### 纯色材质
```javascript
Cesium.Color.RED.withAlpha(0.5)
```

 

#### 图片纹理
```javascript
new Cesium.ImageMaterialProperty({
  image: '/logo-small.svg',
  repeat: new Cesium.Cartesian2(4, 4),
})
```



#### 网格材质
```javascript
new Cesium.GridMaterialProperty({
  color: Cesium.Color.YELLOW,
  cellAlpha: 0.5,
  lineCount: new Cesium.Cartesian2(8.0, 4.0),
  lineThickness: new Cesium.Cartesian2(2.0, 2.0),
})
```



#### 棋盘材质
```javascript
new Cesium.CheckerboardMaterialProperty({
  evenColor: Cesium.Color.WHITE,
  oddColor: Cesium.Color.BLACK,
  repeat: new Cesium.Cartesian2(10.0, 10.0),
})
```



####  条纹材质
```javascript
new Cesium.StripeMaterialProperty({
  orientation: Cesium.StripeOrientation.VERTICAL,
  evenColor: Cesium.Color.WHITE,
  oddColor: Cesium.Color.BLUE,
  repeat: 10.0,
})
```
