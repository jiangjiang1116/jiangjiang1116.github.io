

### 概述
Cesium 是一个基于 WebGL/WebGPU 的开源三维地球可视化引擎，用于构建三维地图、数字孪生、GIS 可视化、仿真等应用。  

Cesium官网：[https://cesium.com/](https://cesium.com/)

Cesium API：[https://cesium.com/learn/cesiumjs/ref-doc/](https://cesium.com/learn/cesiumjs/ref-doc/)

Cesium ion平台：[https://ion.cesium.com/](https://ion.cesium.com/)

### 初始化
利用Vite + Vue创建工程

```shell
npm create vite@latest
```



下载cesium框架

```shell
npm install cesium
```



动态加载静态资源

```javascript
window.CESIUM_BASE_URL = '/Cesium'
```



初始化样式

```javascript
import 'cesium/Build/Cesium/Widgets/widgets.css'
```



设置权限Token

```javascript
Cesium.Ion.defaultAccessToken = 'Cesium ion平台申请'
```



Viewer封装类

```javascript
const viewer = new Cesium.Viewer('cesiumContainer')
```
