

### 概述
CZML（Cesium Language） 是 Cesium 官方定义的一种 JSON 格式的描述语言，用于在浏览器中 动态地展示时序地理数据。

它可以描述：

+ 点 / 线 / 面
+ 模型（gltf/glb）
+ 轨迹（随时间移动）
+ 传感器范围、可视域
+ 标签 / 图标
+ 时间动态属性（位置、方向、颜色、大小…）



```javascript
import pointCZML from './assets/czml/point.czml?url'
import cylindersJSON from './assets/czml/cylinders.json'
import planeJSON from './assets/czml/plane.json'

const dataSource = await Cesium.CzmlDataSource.load(pointJSON)
viewer.dataSources.add(dataSource)

const dataSource2 = await Cesium.CzmlDataSource.load(cylindersJSON)
viewer.dataSources.add(dataSource2)

const dataSource3 = await Cesium.CzmlDataSource.load(planeJSON)
viewer.dataSources.add(dataSource3)

const allEntities = [
  ...dataSource.entities.values,
  ...dataSource2.entities.values,
  ...dataSource3.entities.values,
]

await viewer.zoomTo(allEntities)
viewer.camera.zoomOut(10000)
```
