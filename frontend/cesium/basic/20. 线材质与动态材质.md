

### 线材质
Polyline 不使用普通 MaterialSystem，它有自己的线材质。



####  发光线
```javascript
new Cesium.PolylineGlowMaterialProperty({
  glowPower: 0.2,
  color: Cesium.Color.CYAN
})
```

 

#### 虚线
```javascript
new Cesium.PolylineDashMaterialProperty({
  color: Cesium.Color.YELLOW,
  dashLength: 20
})
```

 

#### 箭头线
```javascript
new Cesium.PolylineArrowMaterialProperty(Cesium.Color.RED)
```



### 动态材质
材质也支持动态属性CallbackProperty。

```javascript
let t = 0
new Cesium.ImageMaterialProperty({
  image: '/public/logo-small.svg',
  repeat: new Cesium.CallbackProperty(() => {
    t = (t + 0.01) % 10
    return new Cesium.Cartesian2(t, t)
  }, false)
})
```
