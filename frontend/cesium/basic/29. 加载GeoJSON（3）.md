

### 概述
GeoJSON 是 Cesium 中最常用的矢量数据格式之一，常用于描述 点、线、面、属性、样式 等信息。  
Cesium 通过 `GeoJsonDataSource` 对其进行解析并以 实体（Entity） 的形式呈现。



```javascript
const handleAdd = async (adcode) => {
  const dataSource = await Cesium.GeoJsonDataSource.load('https://geo.datav.aliyun.com/areas_v3/bound/' + adcode + '_full.json')
  viewer.dataSources.add(dataSource)
  viewer.zoomTo(dataSource.entities).then(() => {
    viewer.camera.zoomOut(10000)
  })
}

const handleRemove = () => {
  viewer.dataSources.removeAll()
  viewer.dataSources.remove(viewer.dataSources.get(0))
}
```
