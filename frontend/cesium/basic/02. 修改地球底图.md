

### 概述
在 Cesium 中，“底图”就是地球表面的影像图层（Imagery Layer），默认为 Bing Maps，来自于Cesium ion平台提供的资源。



### 天地图底图
```javascript
const TDT_TOKEN = '55c0066c8dd3630394c756dfcd97add5'
const tdtImgProvider = new Cesium.UrlTemplateImageryProvider({
  url: `http://t0.tianditu.gov.cn/img_w/wmts?SERVICE=WMTS&REQUEST=GetTile&VERSION=1.0.0&LAYER=img&STYLE=default&TILEMATRIXSET=w&FORMAT=tiles&TILEMATRIX={z}&TILEROW={y}&TILECOL={x}&tk=${TDT_TOKEN}`
})
const tdtCiaProvider = new Cesium.UrlTemplateImageryProvider({
  url: `http://t0.tianditu.gov.cn/cia_w/wmts?SERVICE=WMTS&REQUEST=GetTile&VERSION=1.0.0&LAYER=cia&STYLE=default&TILEMATRIXSET=w&FORMAT=tiles&TILEMATRIX={z}&TILEROW={y}&TILECOL={x}&tk=${TDT_TOKEN}`
})

const viewer = new Cesium.Viewer('cesiumContainer', {
  baseLayer: new Cesium.ImageryLayer(tdtImgProvider),
})
viewer.imageryLayers.addImageryProvider(tdtCiaProvider)
```
