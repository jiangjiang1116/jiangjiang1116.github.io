

### 概述
Cesium 不使用 DataSource 来加载栅格数据，而是通过 影像图层（ImageryLayer） 来加载并呈现。 



```javascript
import TCI from './assets/TCI.tif?url'
import xc from './assets/xc.png'

// 不支持
viewer.imageryLayers.addImageryProvider(new Cesium.SingleTileImageryProvider({
  url: TCI,
  tileWidth: 256,
  tileHeight: 256,
}))

// 支持
let rect = Cesium.Rectangle.fromDegrees(101.73497559, 27.50030558, 102.43557243, 28.18097996)

viewer.imageryLayers.addImageryProvider(new Cesium.SingleTileImageryProvider({
  url: xc,
  tileWidth: 256,
  tileHeight: 256,
  rectangle: rect,
}))

const center = Cesium.Rectangle.center(rect)
const lon = Cesium.Math.toDegrees(center.longitude)
const lat = Cesium.Math.toDegrees(center.latitude)
viewer.camera.setView({
  destination: Cesium.Cartesian3.fromDegrees(lon, lat, 500000),
})
```
