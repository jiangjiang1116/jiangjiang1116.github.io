

### 概述
让相机以某个目标点为中心，用指定的视角和距离进行观察。它会立即把相机切换到一个围绕目标点的相对视图（heading / pitch / range），效果类似定点观察。



```javascript
 viewer.camera.lookAt(
    Cesium.Cartesian3.fromDegrees(121.47, 31.23, 0),
    new Cesium.HeadingPitchRange(
      Cesium.Math.toRadians(0), // 航向角
      Cesium.Math.toRadians(-45), // 俯仰角
      1000000 // 距离
    )
  )
```
