

### 概述
在 Cesium 中，Billboard广告牌代表一个始终面向摄像机的二维图片。



```javascript
const img = new Image()
img.src = logo
img.onload = () => {
  viewer.entities.add({
    position: Cesium.Cartesian3.fromDegrees(121.47, 31.23, 100),
    billboard: {
      image: img,
      width: 70,
      height: 70,
    }
  })
}

viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(121.47, 32.23, 100),
  billboard: {
    image: '/public/logo-small.svg',
    width: 70,
    height: 70,
  }
})

viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(121.47, 33.23, 100),
  billboard: {
    image: 'https://m.360buyimg.com/babel/jfs/t20280615/318148/22/9216/14627/684fdc85Fa154b4e0/0742847dfed41e81.png',
    width: 240,
    height: 70,
  }
})
```



实体模型选中样式。

```javascript
viewer = new Cesium.Viewer('cesiumContainer', {
  selectionIndicator: false, // 关闭点击高亮
  infoBox: false, // 关闭点击弹窗
})
```
