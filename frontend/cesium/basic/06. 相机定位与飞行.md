

### 概述
camera.setView与camera.flyTo分别是进行相机静态定位与动态飞行的实现方法。



### 三维笛卡尔坐标系
三维笛卡尔坐标系，也称三维直角坐标系，是用于在三维空间中确定点位置的坐标系统。它由三条互相垂直的坐标轴（X轴、Y轴、Z轴）组成，它们在原点（O）相交。每个点的位置都由一个有序的三元组 (x,y,z)来表示，其中 x、y、z分别是点到各自对应平面的距离。 

![](https://duyi-resource.oss-cn-beijing.aliyuncs.com/dupeng/1763430415265-c9302afa-e1fe-458d-964f-a7808dde93ae.jpeg)

```javascript
// 经纬度换算笛卡尔坐标系
const xyz = Cesium.Cartesian3.fromDegrees(121.47, 31.23, 100000)
```



camera.setView与camera.flyTo代码示例：

```javascript
viewer.camera.flyTo({
  destination: Cesium.Cartesian3.fromDegrees(121.47, 31.23, 100000),
})

viewer.camera.setView({
  destination: Cesium.Cartesian3.fromDegrees(121.47, 31.23, 100000),
})

viewer.camera.flyTo({
  // 西南角经纬度 + 东北角经纬度
  destination: Cesium.Rectangle.fromDegrees(119, 29, 123, 33), 
})
```
