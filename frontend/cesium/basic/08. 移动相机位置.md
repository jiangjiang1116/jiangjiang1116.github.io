

### 概述
Cesium 的 `Camera` 对象提供了一组 基于当前相机方向 的移动方法，用于实现自由移动视角，类似三维游戏中的 WASD 操作。

| **方法** | **说明（以相机自身坐标系为准）** |
| --- | --- |
| moveForward(amount) | 向前移动（沿视线方向） |
| moveBackward(amount) | 向后移动 |
| moveUp(amount) | 向“相机的上方”移动 |
| moveDown(amount) | 向“相机的下方”移动 |
| moveLeft(amount) | 向左侧水平移动 |
| moveRight(amount) | 向右侧水平移动 |
| lookUp(radians) | 围绕着一个点向上旋转 |
| lookDown(radians) | 围绕着一个点向下旋转 |
| lookLeft(radians) | 围绕着一个点向左旋转 |
| lookRight(radians) | 围绕着一个点向右旋转 |
