

### 概述
可以通过获取控件DOM元素进行位置或样式控制

```javascript
viewer.fullscreenButton.container.style.top = '5px'
viewer.fullscreenButton.container.style.right = '5px'

viewer.homeButton.container.style.left = '5px'
```



可以通过viewModel.command.beforeExecute自定义交互

```javascript
viewer.homeButton.viewModel.command.beforeExecute.addEventListener(function (e) {
  e.cancel = true // 取消默认行为
  viewer.camera.flyTo({
    destination: Cesium.Cartesian3.fromDegrees(121.47, 31.23, 100000),
  })
})
```
