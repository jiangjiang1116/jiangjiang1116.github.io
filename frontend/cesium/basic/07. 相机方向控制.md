

### 概述
在 Cesium 中，相机方向由 heading（航向）、pitch（俯仰）、roll（翻滚） 三个角度共同控制。  
它们决定相机“朝哪里看”，“从什么角度看”，“倾斜程度”。  



![](https://duyi-resource.oss-cn-beijing.aliyuncs.com/dupeng/1763431940669-c5fb67ab-0714-490f-9651-72b51ea15f8c.webp)



```javascript
viewer.camera.setView({
  destination: Cesium.Cartesian3.fromDegrees(121.47, 31.23, 1000000),
  orientation: {
    heading: Cesium.Math.toRadians(0), // 航向角, 默认值为0，即正北方向
    pitch: Cesium.Math.toRadians(-30), // 俯仰角，默认值为-90，即垂直向下
    roll: Cesium.Math.toRadians(30),  // 翻滚角，默认值为0
  }
})
```
