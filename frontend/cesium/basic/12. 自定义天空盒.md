

### 概述
可以通过 scene.skyBox 设置自定义天空盒。

![](https://duyi-resource.oss-cn-beijing.aliyuncs.com/dupeng/1763522349335-fb10397d-c9f4-4c46-832f-72330714ba06.webp)![](https://duyi-resource.oss-cn-beijing.aliyuncs.com/dupeng/1763522387083-4c7b9136-a4da-4170-8aa6-8680092adccb.jpeg)



### 贴图方向
Cesium中的天空盒跟three.js中的天空盒是有图片方向差别的。

![](https://duyi-resource.oss-cn-beijing.aliyuncs.com/dupeng/1763522578465-8fb6c852-85da-42c3-81c3-678d86ade6e7.png)

调整贴图方向

nx：顺时针旋转90°

pz：保持不变

px：逆时针旋转90°

nz：旋转180°

py：保持不变

ny：保持不变

```javascript
viewer.scene.skyBox = new Cesium.SkyBox({
  sources: {
    positiveX: '/skyBox/pos-x.jpg',  // 右
    negativeX: '/skyBox/neg-x.jpg',  // 左
    positiveY: '/skyBox/pos-z.jpg',  // 前
    negativeY:  '/skyBox/neg-z.jpg',  // 后
    positiveZ: '/skyBox/pos-y.jpg',  // 上 
    negativeZ: '/skyBox/neg-y.jpg',  // 下
  }
})
```
