# 空心文字

> 请在深色模式下查看此页面
> text-shadow有八个方向，参考直角坐标系

:::code-group

```css
<h1 style="font-size: 6em;color: #000;">空心文字</h1>
```

```css
<h1 style="font-size: 6em;color: #000;text-shadow:1px 0 #fff">空心文字</h1>
```

```css
<h1 style="
font-size: 6em;
color: #000;
text-shadow:
1px 0 #fff,
1px 1px #fff,
0px 1px #fff,
-1px 1px #fff,
-1px 0px #fff,
-1px -1px #fff,
0 -1px #fff,
1px -1px #fff">
空心文字</h1>
```
